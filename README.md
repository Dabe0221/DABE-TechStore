# DABE TechStore

A modern Spring Boot e-commerce web application built with Java, Spring Boot, Thymeleaf, MySQL, Bootstrap, and Spring Security.

## Features

### Customer
- User registration and login
- Product browsing
- Product search and categories
- Product details page
- Related products
- Shopping cart
- Quantity controls
- Checkout
- Order history
- Order details
- Order status timelineS
- Wishlist
- Product reviews and ratings
- Profile management

### Admin
- Admin dashboard
- Product management
- Product image upload
- Order management
- Update order status
- Revenue analytics
- Weekly revenue chart
- Top selling products
- Highest rated products
- Low stock monitoring
- Recent orders

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Bootstrap 5
- Chart.js
- Maven
- Git / GitHub
